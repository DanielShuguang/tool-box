# 组件开发规范

## Vue 组件规范

1. 使用 `<script setup lang="ts">` 语法
2. 组件文件包含 2 个部分：`.vue`（模板），`logic.ts`（逻辑），如果代码较多，可以拆分为多个文件
3. 使用 Composition API
4. 使用 TypeScript 进行类型定义
5. 使用 Naive UI 组件库
6. 组件拆分：每个功能模块拆分为独立的组件，避免单个组件过于复杂
