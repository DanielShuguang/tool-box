# 技术栈规则

## 项目概述

这是一个基于 Tauri 2.x 的桌面应用，前端使用 Vue 3 + TypeScript，后端使用 Rust。

## 前端技术栈

- **框架**: Vue 3 (Composition API + TypeScript)
- **构建工具**: Vite
- **UI 组件库**: Naive UI
- **CSS 方案**: UnoCSS (原子化 CSS)
- **路由**: Vue Router
- **工具库**: VueUse
- **桌面 API**: Tauri API
- **动画**: motion-v

## 后端技术栈

- **语言**: Rust
- **框架**: Tauri 2.x
- **异步运行时**: Tokio
- **HTTP 客户端**: reqwest
