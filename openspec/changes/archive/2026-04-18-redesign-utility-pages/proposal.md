## 为什么

现有工具页面（翻译、护眼、Windows激活、下载管理、文件搜索、文件读取）视觉风格陈旧，交互体验不一致，部分页面仍使用旧的设计语言。需要统一升级为新的设计语言，与已改造的待办、记账、随机选择等页面保持一致。

## 变更内容

### 页面改造清单

| 页面 | 主要改造内容 |
|------|-------------|
| 翻译工具 | 翻译输入/输出区域、语种选择器、语言切换按钮 |
| 护眼工具 | 定时设置界面、休息提醒弹窗、统计面板 |
| Windows激活 | 激活状态显示、密钥输入区 |
| 下载管理 | 任务列表、进度显示、批量操作 |
| 文件搜索 | 搜索结果列表、高亮匹配、过滤器 |
| 文件读取 | 文件内容展示、编码选择、导航面板 |

### 设计语言统一

- 统一使用 CSS 变量（Naive UI theme vars）
- 统计卡片布局（如待办页面）
- 卡片式容器 + 圆角 + 阴影
- hover 显示操作按钮
- 空状态图标 + 引导文案
- 颜色编码状态指示
