## 新增需求

### 需求：自动保存功能
系统必须支持定时自动保存画布内容到持久化存储。

#### 场景：定时保存
- **当** 自动保存计时器到达设定间隔（5秒）
- **那么** 系统自动将当前画布状态保存到存储中

#### 场景：页面卸载时保存
- **当** 用户关闭页面或浏览器
- **那么** 系统在页面卸载前保存当前画布状态

### 需求：自动保存开关
系统必须提供启动和停止自动保存的能力。

#### 场景：启动自动保存
- **当** 画布初始化完成
- **那么** 系统自动启动自动保存定时器

#### 场景：停止自动保存
- **当** 画布被销毁
- **那么** 系统停止自动保存定时器

### 需求：自动保存数据过期
系统必须自动清理过期的自动保存数据。

#### 场景：数据过期
- **当** 加载自动保存数据时，发现数据超过7天
- **那么** 系统删除过期数据，返回 null
