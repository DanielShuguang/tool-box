## 为什么

在日常生活中经常面临选择困难，例如：
- 和朋友聚餐时不知道吃什么
- 旅游时不确定去哪个景点
- 团队活动时选择地点或项目

需要一个简单易用的随机选择工具，帮助用户快速做出决定。相比传统的抽奖工具，这类场景更注重"随机抽取"而非"中奖"，功能更轻量化。

## 变更内容

1. **新增随机选择工具页面**
   - 创建独立的选择工具页面
   - 支持输入候选项列表进行随机选取

2. **多种选择模式**
   - 普通选择：随机抽取指定数量
   - 顺序选择：按优先级依次抽取（第一选择、第二选择、第三选择等）
   - 权重选择：根据权重概率抽取（偏好某选项时可设置更高权重）

3. **选项管理功能**
   - 已选剔除配置：可选择选中后是否从候选项中移除
   - 禁用功能：支持暂时禁用部分候选项（不参与选取）
   - 禁用名单管理：可批量禁用/启用候选项

4. **结果导出功能**
   - 导出选择结果为 JSON/CSV 格式
   - 支持导出历史选择记录

## 功能 (Capabilities)

### 新增功能

- **random-picker-core**: 随机选择核心功能
  - 随机选取算法实现
  - 支持普通/顺序/权重三种选择模式
  - 支持配置已选项是否剔除
  - 支持禁用部分候选项

- **random-picker-export**: 结果导出功能
  - 导出当前选择结果
  - 导出历史选择记录
  - 支持 JSON 和 CSV 格式
