# 随机选择核心功能

## 概述

实现随机选择工具的核心功能，包括候选项管理、多种选择模式、选项控制和结果导出。适用于日常选择场景，如聚餐选择、旅游目的地选择等。

## 功能规格

### 1. 候选项管理

#### 数据结构

```typescript
interface Option {
  id: string           // 唯一标识
  name: string         // 候选项名称
  weight?: number      // 权重（权重选择时使用，默认为 1）
  disabled: boolean    // 是否禁用
  group?: string       // 分组标识（可选）
}
```

#### 操作

- **添加候选项**: 手动输入单个或批量添加（每行一个）
- **导入候选项**: 从文本文件或剪贴板粘贴导入
- **编辑候选项**: 修改名称、权重、分组
- **删除候选项**: 单个删除或批量删除
- **禁用/启用**: 切换候选项的禁用状态
- **批量禁用**: 选择多个候选项批量禁用/启用

### 2. 选择目标配置（顺序选择模式）

#### 数据结构

```typescript
interface PickTarget {
  id: string           // 唯一标识
  name: string         // 选择目标名称（如"第一选择"、"第二选择"）
  count: number        // 选择数量
  order: number        // 选择顺序
}
```

#### 操作

- 添加/编辑/删除选择目标
- 设置选择数量和顺序

#### 使用场景示例

聚餐时可以这样配置：
- 第一选择：1 个（主选餐厅）
- 第二选择：2 个（备选餐厅）
- 第三选择：3 个（更多备选）

### 3. 选择模式

#### 3.1 普通选择

- 从有效候选项中随机选取指定数量
- 配置项：
  - 选取数量
  - 是否剔除已选项

#### 3.2 顺序选择

- 按目标顺序依次选取（如先选第一选择，再选第二选择）
- 每个目标可配置选取数量
- 配置项：
  - 目标列表（名称、数量、顺序）
  - 是否剔除已选项

#### 3.3 权重选择

- 根据候选项权重计算选取概率
- 权重越高，被选中概率越大
- 使用场景：对某些选项有偏好时，可设置更高权重
- 配置项：
  - 选取数量
  - 是否剔除已选项

### 4. 选项控制

#### 已选项剔除

```typescript
interface PickerConfig {
  removeSelected: boolean  // 选中后是否从候选项移除
}
```

- 开启后，已选项不参与后续选择
- 关闭后，已选项仍可参与后续选择

#### 禁用功能

- 禁用的候选项显示为灰色
- 禁用者不计入有效选项池
- 支持单个禁用和批量禁用
- 支持一键清除所有禁用状态

### 5. 选择结果

#### 数据结构

```typescript
interface PickResult {
  id: string              // 结果 ID
  timestamp: number       // 选择时间戳
  mode: PickMode          // 选择模式
  target?: PickTarget     // 选择目标信息（顺序选择时）
  selected: Option[]      // 被选中的候选项
  config: PickerConfig    // 选择配置
}
```

#### 操作

- 显示选中动画
- 记录选择历史
- 支持撤销最近一次选择（当开启剔除时）

### 6. 结果导出

#### 导出格式

**JSON 格式**:
```json
{
  "exportTime": "2026-03-09T10:30:00Z",
  "results": [
    {
      "timestamp": 1234567890,
      "mode": "sequential",
      "target": { "name": "第一选择", "count": 1 },
      "selected": [{ "id": "1", "name": "火锅" }]
    }
  ]
}
```

**CSV 格式**:
```csv
时间,模式,目标,选中项
2026-03-09 10:30:00,顺序选择,第一选择,火锅
2026-03-09 10:31:00,顺序选择,第二选择,烧烤;日料
```

#### 导出范围

- 导出当前选择结果
- 导出全部历史记录
- 选择时间范围导出

## 技术实现

### 状态管理

使用 Pinia Store 管理以下状态：
- 候选项列表
- 选择目标配置
- 选择配置
- 选择历史
- 持久化存储（使用 tauri-plugin-store 或 Pinia 持久化插件）

### 随机算法

```typescript
// 普通选择：Fisher-Yates 洗牌算法
function randomPick(options: Option[], count: number): Option[]

// 权重选择：加权随机算法
function weightedRandomPick(options: Option[], count: number): Option[]
```
