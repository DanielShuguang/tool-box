## 为什么

当前画板功能仅支持绘制基本图形（如线条、矩形、圆形等），无法插入图片。用户在制作标注、说明或设计稿时，经常需要插入截图、照片等图片素材，这是一个高频使用场景。现在实现此功能可以显著提升画板的实用性和用户体验。

## 变更内容

- 新增图片插入功能，支持多种图片格式
- 新增图片来源支持：本地文件选择、剪贴板粘贴、拖拽上传
- 新增图片操作功能：移动、缩放、旋转
- 新增图片在画板中的渲染和管理逻辑

## 功能 (Capabilities)

### 新增功能
- `canvas-image-insert`: 画板图片插入与管理功能，涵盖图片的插入、渲染和基本操作

### 修改功能
- 无

## 影响

### 受影响的代码
- 前端画板组件：需要新增图片元素类型和渲染逻辑
- 画板状态管理：需要新增图片数据结构和操作方法
- 画板工具栏：需要新增图片插入按钮

### 受影响的 API
- 无新增后端 API，图片处理完全在前端完成

### 受影响的依赖项
- 可能需要引入图片处理库（如用于图片格式解码）

### 受影响的系统
- 画板渲染系统需要支持图片元素的绘制和变换
